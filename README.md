# Análise Exploratória de Dados no Varejo

## Objetivo
Realizar uma análise exploratória de dados utilizando Python e Pandas para identificar problemas de qualidade dos dados, aplicar limpeza e gerar insights sobre vendas no varejo.

## Tecnologias
- Python
- Pandas

## Como executar
1. Instale as dependências:
pip install -r requirements.txt

2. Execute:
python src/pythonMiniProjeto_Turma.py

## Principais etapas
- Importação da base
- Limpeza de Dados:
- Verificação de nulos;
- Tratamento de duplicatas.
- Conversão de datas
- Estatísticas descritivas
